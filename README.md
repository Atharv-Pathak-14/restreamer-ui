# Restreamer-UI

The user interface of the Restreamer for the connection to the Core application.

- React
- Material-UI (MUI)

## Development 

```sh
yarn install
yarn run start
```

Connect the UI to datarhei Core:
http://localhost:3000?address=http://core-ip:core-port/
